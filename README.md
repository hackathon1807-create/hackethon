# OmniCheck 11-11

Multimodal Deepfake Detection MVP for 24-Hour Hackathon.

## Tech Stack
- **Frontend**: React, Vite, shadcn/ui, Tailwind CSS
- **Backend**: FastAPI, Google Gemini 1.5 Flash
- **Analysis**: OpenCV (Visuals), Librosa (Audio)

## Project Structure
- `backend/`: FastAPI server and AI logic.
- `frontend/`: React dashboard.
- `assets/`: Static assets.
